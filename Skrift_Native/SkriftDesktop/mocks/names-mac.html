Mac Names screen — redesigned to match the phone

The Mac's Settings → Names is a plain text list today; this brings it to parity with the phone's Names tab: initials avatars, a voice-enrollment status per person, search, and a person editor with the same fields (Full name · Aliases · Short name · Voice). The Mac has the width, so list + editor sit side-by-side (the Mac idiom) instead of the phone's push-to-detail — same content, same look. Sample roster = your real names DB.
