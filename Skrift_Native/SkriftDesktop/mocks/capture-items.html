Capture items — share a URL / text / image into Skrift, annotate, done

From any app's share sheet → Skrift: a one-glance card — preview of what you're sharing, an optional thought, the significance circles (0 default = stays on the phone, rate it to flag for sync), Save. No recording, no transcription — a capture is the annotation as the note body with the shared thing pinned on top. Shown here as the URL variant; text and image shares swap only the preview block. The circles are live — click them.

Try this for the desktop body editor — the NSTextView part maps onto what [[Nick Jansen]] suggested. Especially the bit on attributed-string round-tripping; check it against our [[link]] styling before committing to the approach.

Flagged choices, easy to flip: the sheet asks for annotation + significance only — no tags, no title (the Mac's Gemma-lite suggests both; a tags row in the sheet would slow the one-glance flow — add one row if that's wrong). Significance lives in the sheet because it's the flag-to-send gate: 0 = the capture stays on the phone, exactly like memos — deferring it to the app would mean every capture needs a second visit before it syncs. The URL card is favicon + title + domain (title from the share payload — no network fetch needed, offline-safe); a plain quoted-URL style is the cheaper fallback. An image capture mirrors the URL flow with a thumbnail as the preview block (and the image travelling as the existing attachments multipart part → an Obsidian ![[embed]] on export); a text capture shows the snippet as a quote and exports it as a blockquote above the annotation. Dictation in the sheet is the small mic on the annotation field — secondary by design (share-sheet extensions get tight memory limits; typing is the primary path, the mic reuses the on-device transcriber if it fits the budget). Contract: a capture uploads through the same POST /api/files/upload with a capture content type + sharedContent/annotationText (already on mobile's UploadMetadata); desktop's UploadService grows a non-audio branch — no transcribe step, body = annotation.
