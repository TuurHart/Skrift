What changed in v2 (from the critique)

  • Read-along contrast fixed — clean 3-step attention ramp (past #6E6E7E → now #F4F4F8 → ahead #A6A6B6); the “ahead = brighter than now” bug is gone.
  • Karaoke word — dropped the filled purple box (it broke reading flow); the active word is now weight-600 + a thin accent underline. No mid-sentence color flip.
  • Tabular numerals everywhere live numbers tick (times, chapter, speed, clock) so the transport doesn’t shimmy.
  • Cover-tint ambiance — the warm cover-derived gradient bleeds down the whole player (the sibling redesign’s best move), and covers are real-object framed (radius + inset hairline + shadow; seeded-hue gradient fallback, no more flat one-offs).
  • Page wins — one 20px column edge, compressed transport (~76px) + tighter header, so the reader gets ≈72–75% of the height. Quiet utility cluster + a single accent Add note chip (your call).
  • Reading mode — now-line pinned to the upper-third focus band (text scrolls under it); “Back to playing” is a transient pill above the float (appears only after you scroll away); auto-recede cue shown (header mid-crossfade).
  • Bookmark ribbon — flush to the page edge, hangs from the top with a swallowtail + soft shadow, plus an 8% accent line-tint so “this bit is saved” actually reads.
  • Accent discipline — the play/float button is the one saturated hero (gradient sphere + glow); scrub/ribbon/track demoted to a muted tint.
  • New states added — not-transcribed nudge + transcribing-in-progress (screen 5), bookmarks empty-state copy (screen 6), and a real delete-confirm with sync-aware copy (screen 7).

Locked decisions (this build)

  • Tab bar: Notes · Library · Highlights(soon) · Settings.
  • “Importance” replaces “significance” (graded, renamed) — lives on the note/memo side, not this player.
  • Add note = accent chip in the utility row.
  • Reading mode = auto-recede after ~3–4s idle + on scroll; tap to show; never while paused; ≈250ms crossfade.
